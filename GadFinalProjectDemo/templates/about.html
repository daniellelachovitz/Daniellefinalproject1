About The Project

The project is a year-end final project within the framework of the "data science" course, of the 10th grade students, in the Tichonet High School in Tel Aviv.

The project combines two main areas of knowledge:

* Web Environment Programming
* Data Science

As part of the project, I located databases on the network and intend to base on them a website that allows access to the databases, data analysis and convenient way of presentation to users. The site will allow the user to explore the data in the chosen field and try to reach valuable insights and conclusions.

The project will be based on the following technological components:

* Python
* Pandas
* Flask
* Bootstrap
* Wtf

The project is based on information and database from the isreali Ministry of Health. The Webstie I created brings the Covid19 cases in simple way that enable the users to understand the infromation in layman terms. The project helped me to develop skills of finding and analyzing information.
